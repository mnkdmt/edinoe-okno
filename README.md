# Запись на приём — г.о. Солнечногорск

Сайт записи жителей на личный приём к руководителям администрации.

## Живой сайт на Render (в один клик)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/mnkdmt/edinoe-okno)

Кнопка откроет Render, прочитает `render.yaml` и развернёт сайт. После сборки
адрес будет вида `https://edinoe-okno.onrender.com`, админка — `/admin/vhod`.

Бесплатный тариф: сервис засыпает при простое (первое открытие ~30–50 сек), база
SQLite сбрасывается при перезапуске. Для постоянной работы — платный тариф с диском.

## Запуск (локально)

    npm install
    npm start
    # открыть http://localhost:3000

Админка: http://localhost:3000/admin/vhod

## Переменные окружения

Проще всего — скопировать образец и заполнить:

    cp .env.example .env

Файл `.env` подхватывается автоматически и в репозиторий не попадает (он в .gitignore).

- `PORT` — порт (по умолчанию 3000)
- `ADMIN_PASSWORD` — пароль входа в админку (по умолчанию `admin` — сменить!)
- `COOKIE_SECRET` — секрет для подписи cookie сессии (сменить в проде!)

## Письма-подтверждения (почта)

Поле «Электронная почта» в форме — по желанию. Если житель его заполнил, после
записи ему уходит письмо с датой, временем и ФИО руководителя.

Отправка идёт по SMTP. Пока `SMTP_HOST` не задан, письма **не уходят**, а пишутся
в лог сервера (запись при этом всё равно создаётся). Чтобы включить реальную
отправку, заполните в `.env` (см. `.env.example`):

- `SMTP_HOST`, `SMTP_PORT`, `SMTP_SECURE`
- `SMTP_USER`, `SMTP_PASS` — логин и пароль ящика-отправителя (для Яндекс/Gmail —
  «пароль приложения», не основной пароль)
- `SMTP_FROM` — адрес/имя в поле «От кого»

Письмо отправляется в фоне и не блокирует запись: если почта недоступна, житель
всё равно записан.

## Данные

SQLite-файл лежит в `data/app.sqlite` (в .gitignore). Каталог тем и руководителей
засевается из `src/data.js` при первом запуске. Чтобы изменить ФИО руководителей —
отредактируйте `src/data.js` и удалите `data/app.sqlite`, затем `npm run seed`.
